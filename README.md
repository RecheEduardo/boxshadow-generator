# Funcionalidades:
<p align="justify">A ideia do site se baseia em uma ferramenta prática para desenvolvedores poderem criar estilizações de sombras nos seus futuros códigos, utilizando-se da propriedade "box-shadow" do CSS, é possível realizar diversas formas diferentes de sombras, com cores, posições e propriedades distintas.</p>
<p align="justify"> Dentro das funcionalidades, vale destacar as funções de mudança dinâmica do código que o programa resulta, sempre de acordo com o exemplo visto pelo usuário durante o uso do site, facilitando o mesmo para ajustar as propriedades com base na sua preferência.</p>

# O que busquei aprimorar nesse projeto:

* Manipulação de métodos do tipo String.

* Estilização e reescrita dinâmica do código com base nas escolhas do usuário.
  
* Utilização de recursos do próprio navegador para acessibilidade do projeto, como a área de transferência onde sua estilização ficará armazenada.

* Aprofundar meu conhecimento sobre os diversos tipos de entradas de dados do usuário no HTML, desde um simples texto até uma seleção de cor ou valor dinâmico.
